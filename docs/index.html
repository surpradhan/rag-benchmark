Key Findings

What we learned

Statistically significant results (Wilcoxon signed-rank, p<0.05) compared against a Basic RAG baseline across 500 questions (3 deterministic runs each, temperature=0).

🏆
Re-ranking wins on precision
Cross-encoder re-ranking achieves the best Recall@5 (0.706) — a statistically significant +15.9% over Basic RAG (p<0.001, d=0.32). The extra latency is justified.
⚡
Hybrid RAG: best coverage, no latency cost
BM25 + vector with RRF achieves Recall@10 of 0.745 (+8.9% over baseline) at nearly identical latency to Basic RAG. The most efficient improvement.
🔮
HyDE is competitive but slow
Hypothetical document embeddings reaches Recall@5 of 0.602 — comparable to Basic RAG — but at 2× the latency due to the hypothesis generation step.
⚠️
Agentic RAG underperforms
The ReAct loop scores −18.2% Recall@5 vs. baseline (p<0.001). Iterative tool use doesn't compensate for context fragmentation on multi-hop questions.
💸
Corrective RAG: expensive, no gain
Re-retrieval logic uses 3× the tokens of Basic RAG and adds significant latency — yet shows no meaningful retrieval improvement (Cohen's d = −0.016).
🕸️
Graph RAG struggles with precision
Community summarisation yields the lowest Recall@5 (0.348), down −42.9% from baseline. Works better at K=10, suggesting broad but imprecise retrieval.

Methodology

Controlled experiment

Every variable except the retrieval strategy is held constant across all runs, ensuring fair comparison.

Dataset

HotpotQA

Distractor dev set — multi-hop questions requiring reasoning across 2+ documents. 500 dev questions, frozen with seed=42.

Embedding Model

all-MiniLM-L6-v2

384-dim sentence embeddings. Fast, free, and well-benchmarked. L2-normalised before indexing for cosine similarity.

Vector Store

FAISS IndexFlatIP

Exact inner product search on L2-normalised vectors — equivalent to cosine similarity. Zero approximation error.

LLM

Llama 3.1 8B (Ollama)

Local inference, zero cost, full reproducibility. temperature=0, seed=42, max_tokens=512 across all patterns.

Chunking

512 tokens / 50 overlap

RecursiveCharacterTextSplitter respecting sentence and paragraph boundaries. Consistent across all patterns.

Statistical Tests

Wilcoxon · Cohen's d

Paired Wilcoxon signed-rank (α=0.05) vs. Basic RAG baseline. Cohen's d for practical effect size — significant but tiny improvements are flagged.
